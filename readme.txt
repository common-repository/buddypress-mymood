== Description ==


BuddyPress MyMood is a BuddyPress, which allows the members to express their moods at different times with a activity status post. Members can use smileys and also show latest mood on the member profile for their friends to know about it.

The plug-in comes with various inbuilt mood types and you can add your custom mood too.

Please comment you view about this plugin here [BuddyPress MyMood Page](http://webgarb.com/buddypress-mymood/).

Note : Highly Recommended to upgrade BP MyMood to 1.2.

== Installation ==

= Automatic Installation =

1. Option on Member Activity Status Post.
2. BuddyPress MyMood On Member Activity Status.
3. BuddyPress MyMood on Member Profile Head.
4. BuddyPress MyMood on Group.
5. BuddyPress MyMood Admin Panel

= Manual Installation =

1. Upload `buddypress MyMood` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
